# Chit Chat

> A simple chatting application using ExpressJs and ReactJs

## Client

SPA using react.

want to spin up just follow

```
    cd client
    yarn
    yarn start
```
[Read More](https://github.com/sujanthakare/chit-chat/blob/main/client/README.md)

## Server

ExpressJS api

want to spin up

```
    cd server
    yarn start:watch
```
